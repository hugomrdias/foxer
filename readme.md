# foxer-repo

> Monorepo for all the foxer packages

## Packages

- [foxer](https://github.com/hugomrdias/foxer/tree/main/packages/foxer) - All-in-one application server and toolkit for Filecoin Onchain Cloud.
- [foxer-client](https://github.com/hugomrdias/foxer/tree/main/packages/foxer-client) - Foxer typescript client.
- [foxer-react](https://github.com/hugomrdias/foxer/tree/main/packages/foxer-react) - React hooks and integrations for building dapps on top of `foxer-client`.
